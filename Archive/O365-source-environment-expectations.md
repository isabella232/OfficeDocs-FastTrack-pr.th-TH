# <a name="source-environment-expectations"></a>ข้อกำหนดของสภาพแวดล้อมต้นทาง
> [!CAUTION]
> เนื้อหานี้จะไม่ใช่เนื้อหาปัจจุบันอีกต่อไปและถูกจัดเวลาให้เอาออก ใช้สารบัญในการนําทางด้านซ้ายมือกับเนื้อหาปัจจุบัน

สิทธิประโยชน์ของFastTrack Center จะให้แนวทางสําหรับคุณในการตั้งค่าระดับการรวมกับสภาพแวดล้อมต้นทางของคุณ (ตัวอย่างเช่น ถ้าคุณมีบริการในสภาพแวดล้อมต้นทางที่คุณต้องการย้ายไปยัง Office 365 อยู่แล้ว)
  
> [!NOTE]
> ถ้าต้องรวม สภาพแวดล้อมต้นทางของคุณต้องอยู่ที่ระดับต่สุดของแอปพลิเคชันนั้น 
  
ตารางต่อไปนี้แสดงความคาดหวังของสภาพแวดล้อมต้นทางที่มีอยู่ของคุณในการออนบอร์ด\*
